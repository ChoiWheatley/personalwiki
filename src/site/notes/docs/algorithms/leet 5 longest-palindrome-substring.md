        ÎÇòÎ®∏ÏßÄÎäî Ïä¨ÎùºÏù¥Îî© ÏúàÎèÑÏö∞Î°ú Ï≤¥ÌÅ¨ÌïòÍ≥†, ÌôÄÏàòÎ°ú Ï∂úÎ∞úÌïòÎäî ÏºÄÏù¥Ïä§ÏôÄ ÏßùÏàòÎ°ú Ï∂úÎ∞úÌïòÎäî ÏºÄÏù¥Ïä§ Î™®Îëê Îã§Ìï¥ÏÑú maxÎ•º Íµ¨ÌïúÎã§.

        Ïôú Solution1Ïóê ÎπÑÌï¥ Îπ†Î•¥ÏßÄ? Îëò Îã§ ÏµúÏïÖÏùò Í≤ΩÏö∞ N**2Ïù∏ Í±∞ ÏïÑÎãåÍ∞Ä? ÎààÏóê Î≥¥Ïù¥Îäî Ï∞®Ïù¥Ï†êÏù¥ÎùºÍ≥§ ÏàòÏ∂ïÍ≥º ÌôïÏû•Ïù∏Îç∞,

        def expand(l: int, r: int) -> str:
            while l >= 0 and r < len(s) and s[l] == s[r]:
                l -= 1
                r += 1
            return s[l + 1 : r]

        if len(s) < 2 or s == s[::-1]:
            return s

        result = ""
        for i in range(len(s) - 1):
            result = max(result, expand(i, i + 1), expand(i, i + 2), key=len)

        return result
