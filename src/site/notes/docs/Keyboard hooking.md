## README

macos에서 하던 양쪽 shift를 동시에 누르면 Capslock이 켜지고 본래 Capslock 버튼은 한영전환키로 만들기 위해 이것저것 알아보고 있다. 아 한 가지 더, ESC 키를 누르면 자동으로 영문으로 돌아오도록 만드는 기능까지.
